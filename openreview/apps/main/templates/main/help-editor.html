Editor help


Headings

  • # Title = First level heading
  • ## Title = Second level heading
  • ### Title = Third level heading

Supported up to six levels.

Emphasis and Importance

  • Emphasis: *example* or _example_
  • Importance: **example** or __example__

Lists

For numbered lists, type 1. followed by a space. Any number (followed by a full stop and space) can be used - all list items will be automatically ordered from 1 on submission.

1. Ordered list item
2. Ordered list item
3. Ordered list item

For bulled lists, type *, - or + followed by a space.

* Bulleted list item
* Bulleted list item
* Bulleted list item

Blockquotes

Type > followed by a space (just like email):

> A quoted paragraph
>> A quoted paragraph inside a quotation

Links

Create a link by surrounding the link text in square brackets, followed immediately by the URL in parentheses:

[text to link](http://example.com/)

Images

Create an image by using the following syntax:

![alt text for image](http://example.com/image.jpg)

Currently only web-accessible image URLs are supported.

Code

You can mark up code in-line using backticks (`code`), or add a code block by adding at least four spaces to the start of a new line:

This is rendered as normal text.
        
    This is a code block 
    
This is rendered as normal text again.

It’s also possible to start a code block with a tab, as long as the text doesn’t start with list, header or blockquote syntax characters (1., *, -, +, #, >).

Note that inline formatting characters (like _underscores_) are ignored in code.

LaTeX

You can use LaTeX formulas inline by wrapping them between $'s. Multi-line LaTeX statements can be shown by wrapping them between $$'s.

For a complete overview of all possibilities with LaTeX, you can visit the LaTeX documentation page.
